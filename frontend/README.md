# Frontend

Next.js frontend for Direct Marketing Contracts portal.

## Setup

```bash
corepack enable
pnpm install
```

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Lint

```bash
pnpm lint
```

## Build

```bash
pnpm build
```

## Start production server

```bash
pnpm start
```
